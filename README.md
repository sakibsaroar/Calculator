This is a multi display realistic animated ui calculator app 
Completely made with JAVA, MAVEN, CSS

HOW TO OPEN

1. Open this repository in github codespace
2. Press crlt + Shift + p and click rebuild container
3. click the file MainApp.java and click run and debug
4. type mvn javafx: run and enter
5. go to ports and open port 6080 for NoVNC in a browser and click connect to use the app in a virtual desktop
